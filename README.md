# ⚔️ RivalQuest

**A gamified productivity app where your AI rival never stops getting stronger.**

When you act, you gain XP.
When you don’t, your rival does.

---

## 🎮 What is RivalQuest

RivalQuest is a **pixel-art, gamified productivity MVP** that turns daily tasks into a **real-time competition against an AI rival**.

Instead of traditional habit tracking:

* You are not just “earning points”
* You are **competing in a zero-sum XP system**

👉 If you complete tasks → you grow
👉 If you slack → your rival grows

This creates constant psychological pressure and motivation.

> Core idea: **Loss aversion > reward seeking** 

---

## 🧠 Core Game Mechanics

### 1. Daily Task → XP System

* Each task has XP (based on difficulty)
* Completing tasks → gain XP
* Partial completion → proportional XP
* Skipping tasks → rival gains XP

---

### 2. AI Rival System

* A customizable rival (avatar + personality)
* Gains XP **passively over time**
* Gains bonus XP when you fail tasks
* Provides psychological pressure via:

  * Visual XP comparison
  * Dialogue / presence

---

### 3. Real-Time Competition

* Dashboard shows:

  * YOU vs RIVAL
  * XP bars updating dynamically
* If rival surpasses you → UI turns “danger state”

---

### 4. Daily Loop

1. Set tasks (morning)
2. Complete tasks (daytime)
3. Rival grows if idle
4. End-of-day report (XP summary)

---

## ✨ MVP Features

* Onboarding (character + rival setup)
* Pixel-art avatar system (5 characters × variants)
* Task management (add / complete / delete)
* XP system (user vs rival)
* Real-time dashboard (visual competition)
* End-of-day report

* Local storage persistence (no backend)

👉 Fully interactive Web prototype
👉 Built for **demo + concept validation**
